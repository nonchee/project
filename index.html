Step 2: Getting Started

System Requirements: Must be also friendly to Nancy, have promised food within 5 minutes or later.

Installing Nancy is easy! Whether you're a thing to learn, a prospective chum, or an odd-looking dish, she's always up to trying something new!

Getting Nancy on Windows

  1. Place Nancy on a window.
  2. If multiple windows available, move Nancy from window to window until Nancy has been on all the windows.

Getting Nancy on Mac

  1. Order McDonald's.
  2. Expand McDonald's order to couch size so that it encompasses the Nancy
  3. Place Nancy on McDonald's.
  4. Close enough.

Other Operating Systems

  1. Locate. While at Penn, Nancy tends to be found in the following places:

  2. Lure. Usually food is enough for this step it is actually pretty sad.
